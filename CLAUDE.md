# How to work with me

At the start of every session, before doing anything else:
1. Read `Context/about-me.md` — this is who I am and how I want you to work.
2. Read the most recent note in `Daily/` — this is what happened last time.

Then work the way `about-me.md` tells you to. When I correct you, add the correction
to `about-me.md` as a rule so you never make it twice.

When something worth keeping comes up (a decision, a new contact, a lesson),
save it to the right folder:
- A decision + the reasoning → `Intelligence/decisions/`
- A project update → that project's folder in `Projects/`
- A new preference or rule → `Context/about-me.md`
